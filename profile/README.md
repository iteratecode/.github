### Code View

![Code Wireframe](https://raw.githubusercontent.com/iteratecode/.github/main/images/code-wireframe.png)

```
@startsalt
{S
01. /* Return the sum of the input */
02. public int Sum (int[] input)
03.   int total = 0;
04.   for (int i = 0; i < input.Length(); i++)
05.     total += input[i];
06.   return total;
}
@endsalt
```
